Установка
---------

Для установки модуля вам нужно только включить модуль как при стандартной установке модуля.




Страница курса валют
--------------------
Страница курса валют с сортировкой и фильтрацией по дате обновления курса валют и дате доступна по адресу: admin/custom_exchange_rates.




Обновление курса валют
----------------------
Курс валют обновляется в двух режимах:

- Автоматический (по запуску крона)
- Ручной (по ссылке admin/custom_exchange_rates/update)




Данные в виде JSON
------------------
Отфильтрованные данные курса валют можно скачать в виде .json по ссылке admin/custom_exchange_rates/json




Права и разрешения
------------------

Для того, чтобы пользоваться функционалом модуля, необходимо включить Права пользователя:

"custom exchange rates administrator" - Администрирование курса валют
